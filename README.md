# Simple Text Editor

A simple text editor built using HTML, CSS, and JavaScript.

## Features

- **Undo/Redo:** Easily undo or redo your actions.
- **Font and Font Size:** Customize the font and size of your text.
- **Font Color:** Choose the color of your text.
